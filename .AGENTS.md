Offr3d Agent Quickstart
=======================

Context
-------
- **Domein**: `https://xinudesign.be/offr3d` (HTTPS wordt afgedwongen via `.htaccess`).  
- **Frontend**: React 18 + Vite + Tailwind CSS.  
- **Styling**: `terminal-*` utility classes (zie `src/index.css`) + theme tokens in `tailwind.config.js`.  
- **Backend**: PHP endpoints onder `public/api`, MySQL via PDO (`db.php`).  
- **State**: `SettingsContext` + `ToastContext` voor settings en meldingen.  
- **Sitemap**: `public/sitemap.xml` bevat de belangrijkste routes en wordt gepubliceerd met de build.

Local Commands
--------------
- Install deps: `npm install`
- Start dev server: `npm run dev`
- Build production bundle: `npm run build`
- Preview production build: `npm run preview`

Key Paths
---------
- Dashboard: `src/pages/Home.jsx`
- Nieuwe/bewerk offerte: `src/pages/NewQuotePage.jsx`, `src/components/PrintItemList.jsx`, `src/components/QuoteForm.jsx`
- Offerte-overzicht/detail: `src/pages/QuotesPage.jsx`, `src/pages/QuoteDetailPage.jsx`
- Settings: `src/pages/SettingsPage.jsx`, `src/context/SettingsContext.jsx`
- Materialen: `src/components/settings/MaterialManager.jsx`
- Meldingen: `src/context/ToastContext.jsx`
- Routing/meta: `index.html`, `.htaccess`, `public/sitemap.xml`

Working Guidelines
------------------
1. **Styling**  
   - Gebruik bestaande kleuren (`primary`, `accent`, `parchment`, `ink`, `gridline`, `signal.*`).  
   - Houd `terminal-card`, `terminal-button`, `terminal-pill`, `terminal-note` consistent.  
   - Bewaar de neo-retro look: matte beige/charcoal, monospaced fonts, subtiele animaties.

2. **Data Flow**  
   - Quote berekeningen: `calculateItemCost.js` → `calculateQuoteCost.js`.  
   - Settings laden/opslaan via `SettingsContext` + `update-settings.php`.  
   - Offertes CRUD: `save-quote.php`, `update-quote.php`, `delete-quote.php`, `get-quote-detail.php`.  
   - Dashboarddata: `/api/get-quotes.php`, `/api/get-materials.php`, `/api/get-clients.php`.

3. **Deployment Notes**  
   - HTTPS moet altijd actief blijven; voeg redirects toe via `.htaccess` wanneer nieuwe mappen ontstaan.  
   - Werk `public/sitemap.xml` bij als er nieuwe publieke routes bijkomen.  
   - Metadata (titel/description) staat in `index.html`; houd naam “Offr3d | XinuDesign” aan.

4. **When in doubt**  
   - Check deze README en `.AGENTS.md` voor context.  
   - Draai `npm run build` voor regressiecheck.  
   - Volg bestaande endpoint-structuur in `public/api` bij uitbreidingen.
